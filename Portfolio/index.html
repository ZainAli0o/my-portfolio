Work Experience

HTML
HTML (HyperText Markup Language) is the core language for building web pages. It structures content with elements like headings, paragraphs, and links, forming the foundation of web development.
CSS
CSS (Cascading Style Sheets) is used to style and format web pages. It controls layout, colors, fonts, and responsiveness, enhancing HTML's visual presentation, ensuring consistency, and improving user experience across devices.
Javascript
JavaScript is a powerful programming language for creating interactive web pages. It enables dynamic features like animations, form validation, and content updates, enhancing user experience. Widely used, it works with HTML and CSS.
React js
React is a JavaScript library for building user interfaces. Developed by Facebook, it enables the creation of reusable UI components, ensures efficient rendering with a virtual DOM, and simplifies building dynamic, responsive web applications.
Node Js
Node.js is a JavaScript runtime built on Chrome's V8 engine. It allows developers to run JavaScript on the server side, enabling scalable, fast, and efficient backend development for web applications.
Mongo DB
MongoDB is a NoSQL database that stores data in a flexible, JSON-like document format. It supports scalability, high performance, and dynamic schemas, making it ideal for modern applications requiring unstructured or semi-structured data.
